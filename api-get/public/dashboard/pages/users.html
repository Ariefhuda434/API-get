Managed Users

Create ghost users and generate access tokens for Klap embed

How it works

1. Click "Create User" to create a managed sub-user under your Klap account.
2. Generate an access token for that user to authenticate the Klap embed player.
3. Use the token to embed the Klap editor in your dashboard or share it with clients.

All content created by managed users is billable to your Klap account.
